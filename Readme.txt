FloodIt game

To play the flood it game you are required to have all the supporting documents in the same directory.

type im make run to run the game then repond appropriately to the prompts
i.e
make run
false
false

this will just start the standard FloodIt game (15 x 15 grid)

/****************CHANGING GRID SIZES***********************/
make run
false
true

/***************CONTINUE FROM LAST SAVED POINT*************/
make run
true

/**************THE COMPUTER PLAYING THE GAME****************/
make solve

have fun :)
